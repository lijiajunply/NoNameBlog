# 前言
对于经常关注数码科技圈（喜欢白嫖付费软硬件）的人来说，拥有一个校园邮箱有多么重要。
举个例子，GitHub学生包里头包含有：
- **GitHub Pro** 和 **GitHub Team**
- **云服务** 如 AWS、Azure 和 DigitalOcean
- **开发工具** 折扣，如 Atom、SQL Server 和 Namecheap 域名
- **教育课程**，提供60多门课程，涵盖热门话题，帮助学习编程和技术面试准备
- **AI Prompting & Technical Writing**：包括 GitHub、Microsoft Azure、Educative 和 POEditor 等工具
- **Security and Monitoring**：包括 GitHub、Blackfire、Honeybadger 和 AstraSecurity 等安全和监控工具
- **Career Readiness**：包括 Educative、InterviewCake、DailyBot 和 GitHub Pages 等职业准备工具
- **Intro to Web Dev**：包括 LambdaTest、Polypane、Bootstrap Studio 和 DigitalOcean 等网页开发工具
等等等等。而这个GitHub学生包就必须要使用到校园邮箱才可以注册使用。
还有就是著名IDE生产企业JetBrains。或许你们没有听过这个公司，但是你们肯定接触或者以后会接触到他的产品，例如写Java必备的Idea和用于写Python的PyCharm，这些优秀产品都是他们家做的。
![](/notion-images/xauat-1/a767e6ba3e34.png)
是的，他们家也可以进行学生认证并且可以免费使用他们家的全部IDE软件(集成开发环境)，包括专业版和社区版，这些都是完全可以使用的。
建大为了让我们白嫖到国外优秀资源，师夷长技以制夷，也是贴心的为我们弄了校园邮箱网站。我们也得明白学校的良苦用心呀！
我们开始吧。
# 创建一个校园邮箱账号
1、 首先进入到咱们的**西安建筑科技大学综合服务平台 (https://ehall.xauat.edu.cn)**
![](/notion-images/xauat-1/001-dc01b85f67fb.webp)
**2、 **登录你的西建大综合服务平台账户**（用户名为工号/学号，初始密码为本人身份证后六位）**
![](/notion-images/xauat-1/002-57e1eb497145.webp)
**3、 **点击右上角的“校园邮箱申请”
![](/notion-images/xauat-1/003-fc80b3c89963.webp)
**4、 **点击“**立即办理**”
![](/notion-images/xauat-1/004-e2569802c263.webp)
**5、 **按照要求输入申请人信息，点击**确认**
![](/notion-images/xauat-1/005-d6601369f9c6.webp)
**6、 **静候**1\~2天**，等待我校信息办审核好就可以啦！

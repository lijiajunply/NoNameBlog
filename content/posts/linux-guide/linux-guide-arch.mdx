### **二 配置**
### **0 关于manjaro最新版没有网卡驱动的问题（2022.11.16）**
联想小新pro16（2022版）的网卡比较新，而manjaro最新版现在内核版本才5.15，不支持
更新到5.19就行了

### **1 yay 和 paru**

```shell
sudo pacman -S yay
sudo pacman -S paru
```

### **2 换成国内源**

```shell
 sudo pacman-mirrors -i -c China -m rank # 更新镜像源排名，选择延迟小的
 sudo nano /etc/pacman.conf
```

在文件末尾写上：

```shell
[archlinuxcn]
Server = https://mirrors.ustc.edu.cn/archlinuxcn/$arch
```

其实清华的也不错，但是我更推荐中科大的，因为清华的东西少，而且有点慢
清华的：

```shell
[archlinuxcn]
Server = https://mirrors.tuna.tsinghua.edu.cn/archlinuxcn/$arch
```

保存之后在终端上输入：

```shell
sudo pacman -Syy && sudo pacman -S archlinuxcn-keyring
```

当然你也可以试试其他的源：
[GitHub - archlinuxcn/mirrorlist-repo: Arch Linux CN Community repo mirrors list](https://link.zhihu.com/?target=https%3A//github.com/archlinuxcn/mirrorlist-repo)

### **3 输入法**
直接：

```shell
yay -S fcitx-sogoupinyin
```

就行了
比Ubuntu简单多了

### **4 安装腾讯系软件**
4.1 tim

```shell
yay -S deepin-wine-tim
```

4.2 qq

```shell
yay -S deepin-wine-qq
```

4.3 微信

```shell
yay -S deepin-wine-wechat
```

### **5 wps**
先安装字体：

```shell
yay -S wps-office wps-office-mui-zh-cn wps-office-fonts ttf-wps-fonts
yay -S ttf-ms-fonts
```

再安装软件：

```shell
yay -S wps-office-cn
```

# 打开限制广告跟踪

设置→权限与隐私→设备标识和广告→打开限制广告跟踪

然后一个一个的关掉就行了

# 关闭用户体验计划

设置→关于本机→改进体验计划→关闭加入用户体验计划

# 关闭乐划锁屏

设置→桌面与锁屏→乐划锁屏→关于乐划锁屏→关闭乐划锁屏

# 关闭小布建议广告

右滑负一屏→设置→小布建议→关闭建议

# 关闭主页卡片广告

右滑负一屏→点击搜索框→设置→主页卡管理→搜索历史应用建议热门搜索

# 关闭我的oppo广告

我的oppo→我的→右上角功能项→关闭不必要的功能

# 关闭浏览器广告

浏览器→我的→设置→隐私→关闭个性化推荐

# 关闭视频app广告

视频→我的→设置→关闭个性化推荐

# 关闭主题商店广告

主题商店→我的→右上角设置→消息与推送设置→关闭消息通知、个性化推荐设置

# 关闭软件商店广告

软件商店→我的→消息管理→全部关闭

软件商店→我的→服务管理→全部关闭

软件商店→我的→隐私→个性化推荐→全部关闭

# 关闭钱包广告

钱包app→我的→设置→隐私设置与服务管理→个性化推荐设置→关闭【个性化推荐】

钱包app→我的→设置→隐私设置与服务管理→功能项→关闭全部开启
